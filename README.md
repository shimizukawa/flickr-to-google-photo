# Flickrの写真をGoogleに移行するスクリプト

## やりたいこと

- Flickrからプライベート設定の写真と、メタ情報のアルバム、タイトル、コメント、撮影日時、GPS情報、他取得できるメタデータを取ってくる
- Google Photo に写真をアップロードしてアルバム情報、タイトル、コメント、他つけられる情報を付ける
- メタ情報と移行状況だけをローカルにJSONで保存する
- 各写真は、Flickrからダウンロード、GooglePhotoへアップロード、Flickrから削除、でワンセット
- メタ情報には、写真ごとにFlickrのURL、GooglePhotoのURL、移行状況、アルバム名、タイトル、コメント、撮影日時、GPS、他あるとよさそうな情報を保存する
- Googleアップロード前に写真画像ファイルにメタデータを付与する必要があるかもしれない
- Flickrからの写真ダウンロード時は一番解像度の高いデータを取得する
- GooglePhotoへの写真アップロード時は、GooglePhotoの最適サイズ（APIで指定できるなら）を指定する
